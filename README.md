# How to Use
# 1.After pressing any key to start, enter a number between 1-6 to select a cinema schedule, or enter `search` to check movie reviews.


![image](https://github.com/jim0987795064/lineBot/blob/main/t4.jpg)
# 2.After selecting a number, the schedule for the selected cinema will be displayed.
![image](https://github.com/jim0987795064/lineBot/blob/main/t3.jpg)
# 3.Enter `book` to go to the ticket booking link.
![image](https://github.com/jim0987795064/lineBot/blob/main/t2.jpg)
# 4.Enter `search` to check movie reviews.
![image](https://github.com/jim0987795064/lineBot/blob/main/t1.jpg)
